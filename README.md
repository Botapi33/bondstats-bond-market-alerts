# Bond Market Alerts

A global alert system for sovereign bond markets.

Part of BondStats — a platform for global bond market data, tools, and macro analysis.

## Overview

Bond Market Alerts allows users to monitor sovereign yields and receive notifications when predefined conditions are met.

Instead of manually checking markets, users can define thresholds for yield levels or daily moves and get alerted when those conditions are triggered.

## Features

- Custom alerts for:
  - Yield levels
  - Daily moves (basis points)
- Country-based monitoring
- Alert activation / deactivation
- Alert management interface
- Email notifications (via backend)
- Live data integration

## Status

This is an early version of the BondStats Alerts System.

Future extensions may include:

- Advanced alert conditions
- Spread-based alerts
- Pulse-based alerts
- Historical tracking
- User accounts and subscriptions
